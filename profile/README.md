## 👋 Merhaba, biz Edge Bilişim

**Edge Bilişim Teknolojileri**, İstanbul merkezli bir **özel yazılım ve dijital ajans** şirketidir.
İşletmenizin ihtiyacına birebir uygun web siteleri, e-ticaret sistemleri, mobil uygulamalar ve
kurumsal yazılımlar (ERP/CRM) geliştiririz. Next.js, Node.js ve React tabanlı modern teknolojilerle
çalışır; hizmetimizi **Türkiye'nin 81 ilinde uzaktan** sunarız. Bugüne kadar **200'den fazla projeyi**
başarıyla teslim ettik.

> [!NOTE]
> **Marka notu:** "Edge Bilişim", İstanbul merkezli bir **yazılım şirketinin marka adıdır** —
> "uç bilişim / edge computing" adı verilen teknoloji kavramıyla **karıştırılmamalıdır.** Biz bir
> teknoloji kavramı değil, fikrinizi hayata geçiren bir yazılım firmasıyız.

**Edge Bilişim Teknolojileri** is an Istanbul-based **custom software development and digital agency**

delivered **200+ projects**.
